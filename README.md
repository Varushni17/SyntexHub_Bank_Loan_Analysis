Bank Loan Analysis

Project Overview
This project analyzes bank loan data to identify trends, calculate key performance indicators (KPIs), classify loans into good and bad loans, and visualize loan performance.

Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

Dataset
- Lending Club Loan Dataset
- Records: 39,717
- Features: 111

KPIs
- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income (DTI)
- Good Loan Percentage
- Bad Loan Percentage

Visualizations
- Loan Status Distribution
- Loan Grade Distribution
- Loan Purpose Distribution
- Top States by Loan Applications
- Monthly Loan Applications
- Interest Rate Distribution
- Annual Income Distribution

Conclusion
The analysis shows that 85.83% of loans are Good Loans (Fully Paid or Current), while 14.17% are Charged Off. The notebook provides insights into loan performance and borrower characteristics through KPIs and visualizations.
